# Glowera Product Rules

## Design Philosophy
- calm
- premium
- feminine
- minimal
- emotionally supportive

## Avoid
- AI slop UI
- clutter
- generic card spam
- childish gamification

## UX Principles
- whitespace first
- emotional UX > productivity UX
- subtle animations only
- ritual-based flows

## Visual Style
- soft cream backgrounds
- blush/sage/lavender palette
- elegant serif headings
- no harsh contrast

## Product Direction
Glowera is a wellness ritual platform for women.
Not a productivity app.
Not a gamified app.
